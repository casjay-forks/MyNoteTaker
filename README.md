# MyNoteTaker

## Introduction

This application can be used to write, save, and delete notes. This application will use an express backend and save and retrieve note data from a JSON file. 

## Technologies Used
    * HTML
    * CSS
    * Bootstrap CSS
    * JavaScript
    * NodeJS
    * Express
    * jQuery
